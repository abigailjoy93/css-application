CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to create a Flexbox row layout

.row {
    display: flex;
    flex-direction: row;
}

Flexbox Column

Use these properties to create a Flexbox column layout.

.column {
    display: flex;
    flex-direction: column;
}

CSS Media Query

Make a responsive webpage based on the screen size.

@media screen and 
(max-width: 992px) {
    header {background: white;}
}

@media screen and 
(max-width: 768px) {
    header {background: yellow;}
}

@media screen and 
(max-width: 576px) {
    header {background: blue;}
}

Linear Gradients

This will create a background linear gradient from top to bottom.

.linear-gradient-background {
    background image: 
    linear-gradient
    (0deg, gray, black);
}

Box Transition

Use transitons and box shadows to glow on hover.

.box-glow:hover {
    box-shadow: 
    15px 15px 10px 3px purple;
    transition-duration: 2s;
    transition-timing-function: 
    linear;
}

Raising Elements

Use this to raise elements higher than their natural starting point.

.card-header {
    position: relative;
    margin-top: -20 px
    }
